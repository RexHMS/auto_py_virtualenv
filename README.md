# auto_py_virtualenv
以後執行這個檔案  
1.會自動到D 槽  
2.檢查D 槽有沒有Python資料夾，沒有就建立 並移動到 Python資料夾下  
3.等待你輸入虛擬環境名稱 abc  
4.建立資料夾 abc  
5.移動到資料夾 abc 下 

**注意**  
這邊 %pip install virtualenv%  
如果沒安裝虛擬庫 virtualenv  
建議先安裝
不然就是要把這行的兩個 % 都拿掉，讓它安裝一次  

6.建立虛擬環境 abc  
7.啟動虛擬環境 abc  

然後你就可以去打開  
VS code、Jupyter那些編譯器，  
指到Python\abc，  
就有一個乾淨的虛擬環境abc，  
就可以開始打 code
